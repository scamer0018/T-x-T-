# 📥 Installation from Your Repository

For Users (One Line Install):

```bash
curl -sL https://raw.githubusercontent.com/scamer0018/T-x-T-/main/install.sh | bash
```

Manual Installation:

```bash
# Clone repository
git clone https://github.com/scamer0018/T-x-T-.git
cd T-x-T-

# Make installer executable
chmod +x install.sh

# Run installer
./install.sh
```
