# 📥 Installation from Your Repository

For Users (One Line Install):

```bash
curl -sL https://raw.githubusercontent.com/scamer0018/termux-pro-theme/main/install.sh | bash
```

Manual Installation:

```bash
# Clone repository
git clone https://github.com/YOUR_USERNAME/termux-pro-theme.git
cd termux-pro-theme

# Make installer executable
chmod +x install.sh

# Run installer
./install.sh
```
